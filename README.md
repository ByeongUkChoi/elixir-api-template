# Elixir API server
## spec
- elixir
- phoenix framework

## Elixir Project
### install elixir for mac
```
brew install elixir
```
### create project
```
mix new example
```
### complie project
```
mix compile
```
### run project (interactive mode in iex)
```
iex -S mix
```
#### check & call function
```
iex > Application.loaded_applications
iex > ElixirExampleProject.hello
```
### recompile in iex
```
iex > recomple
```
### run project
```
mix run
```

## Phoenix framework Project

### install phoenix
```
mix archive.install hex phx_new
```
### create elixir project
```
mix phx.new demo --live
```
